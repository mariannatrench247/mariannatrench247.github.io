Deep Learning Models

In this project, I worked on developing and training deep learning models using PyTorch and TensorFlow. The models are trained on large datasets to perform various tasks like image classification, object detection, and more.

Key highlights of this project:

  • Built and trained foundation models on large datasets.
  • Optimized neural networks using GPUs and CUDA programming.
  • Implemented SLAM for robotics and object detection algorithms.
